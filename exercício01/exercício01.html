Como fazer o processo de abrir o seu arquivo do GIT HUB até o Visual Studio Code

1- Abre o GIT BASH.

2- Vai no site "git config", vai na parte de "sua identidade" e la você vai encontrar 2 códigos:

PRIMEIRO CÓDIGO: "$ git config --global user.name "Fulano de Tal" (o "$" você não copia), e em vez de "fulano de tal", você colocará o seu nome no GIT e clica "enter";

SEGUNDO CÓDIGO: "$ git config --global user.email fulanodetal@exemplo.br" (o "$" você não copia), e faz o mesmo processo do primeiro.

3- Vai no GIT HUB, acha o seu repositório, clica em "CODE" e copia.

4- volta no GIT BASH e cola e clica "enter".

5- Digita "git clone e coloca o seu arquivo.
